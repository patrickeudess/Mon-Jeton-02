🔒 Vos données

Vous gardez le contrôle sur vos informations et sur ce qui est partagé avec votre groupe.

Ce que fait Mon Jeton

Mon Jeton aide à suivre un budget, une tontine ou une AVEC. L’application ne reçoit pas l’argent et ne réalise aucun transfert.

Pour les groupes : les noms des membres, les cotisations, les demandes de prêt et le journal du groupe sont partagés seulement avec les membres autorisés et connectés.

Données enregistrées

  • Sur cet appareil : budget, transactions, objectifs et réglages.
  • Avec un compte connecté : ces données personnelles peuvent être synchronisées avec Firebase.
  • Dans un groupe : les informations nécessaires au suivi collectif sont visibles par les membres de ce groupe.

Ne partagez jamais votre mot de passe ni le code d’accès d’un groupe avec une personne non concernée.

Vos choix

Vous pouvez télécharger une copie de vos données personnelles ou les supprimer du compte connecté. La suppression de vos données personnelles ne supprime pas l’historique déjà partagé dans les groupes : le responsable du groupe doit gérer ce suivi avec les membres.

Besoin d’aide ?

Pour un problème d’accès, de groupe ou de données, décrivez le souci dans le dépôt de support Mon Jeton.
