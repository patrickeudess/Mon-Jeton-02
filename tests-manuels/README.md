# Tests manuels

Ce dossier regroupe les pages de test et de vérification manuelles qui se
trouvaient auparavant à la racine du projet. Elles ne sont pas chargées par
l'application en production.

Note : ces pages référencent les pages de l'application par chemin relatif
(ex. `savings.html`). Depuis ce dossier, ouvrez-les via un petit serveur local
lancé à la racine du projet (`python -m http.server 8080`) puis préfixez les
liens cassés par `../` si nécessaire.
